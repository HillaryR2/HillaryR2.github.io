Project #2: My Fantastic Beast

Project Statement #2:

To come up with my animals I decided that I wanted my mythical creature to be fast, have wings, and have good senses. My animal has the ability to fly, run fast, and it can also smell, hear, and see anything within a 1 mile radius. To create my creature I looked up 3 different animals which are: a tiger, eagle and seabird. After that I used Adobe Photoshop to merge my 3 animals and create my mythical creature. It has the body of a tiger, the head of an eagle, and wings of a seabird.

Project #3: Making Mandalas

Project Statement #3:

I made this project using Adobe Photoshop where I chose the type of flower I wanted, which was a rose. I then placed 8 roses and formed a circle with them. After I chose the placement and type of flower I wanted, I repeatedly pasted the same 8 roses to make the circle smaller and rotated it a couple of degrees each time. The colors I used are dark and light red, black, and gray. What I noticed about repeating the same steps is that I was able to create a spiral-like illusion.

Project #4: My Pixel Art Character

Project Statement #4:

I used Adobe Photoshop to create my pixel character. I started by looking for the right head size that would allow me to have the face features I wanted and what size the body would be. My character is a female wearing a black dress, red heels, gold earrings, and long red hair. My character would be in a game where she fights against crime and bad guys. Her purpose in the game is to be undercover and investigate the bad guys.

Project #5: What's Your Vibe?

Project Statement #5:
